# 🌐 AI-Портфолио: Сайт-визитка с RAG-ассистентом

Профессиональный сайт-визитка AI-разработчика с интегрированным умным чатботом на базе технологии RAG

## 🎯 Обзор проекта

Полный стек веб-приложения, включающий:
- **Современный сайт портфолио** с адаптивным дизайном
- **Админ-панель** для управления контентом без кода
- **AI-ассистент** с технологией RAG (Retrieval-Augmented Generation)
- **Telegram-бот** для уведомлений о новых сообщениях

## 🚀 Демонстрация

### 🌐 Веб-сайт
- **Современный дизайн** в AI-тематике с плавными анимациями
- **Адаптивная верстка** для всех устройств (desktop, tablet, mobile)
- **Чат-виджет** с умным AI-ассистентом

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/holmok1577-ops/portfolio-website.git
cd portfolio-website
```

### 2. Создание виртуального окружения
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
Создайте файл `.env` в корне проекта:
```env
# OpenAI API ключ (основной)
OPENAI_API_KEY=your_openai_api_key_here

# Или ProxyAPI ключ (альтернатива для РФ)
PROXY_API_KEY=your_proxy_api_key_here

# Telegram бот (опционально)
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_chat_id

# Настройки сервера
HOST=0.0.0.0
PORT=8000
```

### 5. Запуск приложения
```bash
python main.py
```

Приложение будет доступно по адресу: `http://localhost:8000`

## 🚀 Использование

### 🌐 **Работа с сайтом**
1. Откройте `http://localhost:8000` в браузере
2. Навигация по разделам портфолио
3. Чат с AI-ассистентом в правом нижнем углу
4. Админ-панель доступна по `/admin`

### ⚙️ **Админ-панель**
1. Перейдите в `/admin`
2. Введите пароль (по умолчанию: `admin123`)
3. Управляйте контентом, услугами, проектами
4. Настройте чатбота и системные параметры

### 🤖 **RAG-ассистент**
1. Задайте вопрос в чат-виджете
2. Система найдет релевантные документы
3. Получите умный ответ с учетом контекста

## 🛠️ Кастомизация

### 🌐 **Изменение контента сайта**
Отредактируйте данные в админ-панели или в `frontend/script.js`:
```javascript
// Изменение информации о себе
const aboutData = {
    name: "Ваше имя",
    title: "Ваша должность",
    description: "Описание"
};
```

### 🤖 **Добавление документов в RAG**
В файле `embeddings.py`:
```python
def get_sample_documents():
    return [
        ("Название документа", "Содержание документа..."),
        ("Еще документ", "Дополнительная информация...")
    ]
```

### ⚙️ **Настройка Telegram уведомлений**
1. Создайте бота в Telegram через @BotFather
2. Получите токен и ваш chat ID
3. Добавьте в `.env` файл
