## 📋 Описание проекта

Это полнофункциональный портфолио-сайт, включающий:

- **🏠 Современный сайт-визитка** с адаптивным дизайном
- **🤖 RAG-ассистент** на базе FastAPI для ответов на вопросы по AI
- **⚙️ Админ панель** для управления контентом сайта
- **📚 База знаний** по LLM моделям, RAG системам и промпт-инжинирингу

### 🎯 Возможности

**Сайт:**
- Hero секция с анимациями
- Раздел "Обо мне"
- Услуги и проекты с изображениями
- Контактная информация
- Интегрированный чат-бот

**RAG-ассистент:**
- Ответы на основе локальной базы знаний
- Контекстно-зависимые ответы
- Быстрый поиск по документации

**Админ панель:**
- Управление всем контентом сайта
- Загрузка изображений для проектов
- Настройка приветственного сообщения чат-бота
- Смена пароля администратора

## 🛠️ Стек технологий

### Backend
- **FastAPI** - современный веб-фреймворк для Python
- **ChromaDB** - векторная база данных для RAG
- **Sentence Transformers** - эмбеддинги для текста
- **Pandas** - обработка данных

### Frontend
- **HTML5/CSS3** - семантическая верстка и стилизация
- **Vanilla JavaScript** - интерактивность без фреймворков
- **Font Awesome** - иконки
- **Google Fonts** - типографика

### AI/ML
- **RAG (Retrieval-Augmented Generation)** - архитектура ассистента
- **Vector Search** - семантический поиск
- **Local Embeddings** - приватность данных

## 🚀 Установка и запуск

### Требования
- Python 3.8+
- Git

### 1. Клонирование репозитория
```bash
git clone https://github.com/holmok1577-ops/portfolio-website.git
cd portfolio-website
```

### 2. Установка зависимостей
```bash
pip install -r backend/requirements.txt
```

### 3. Запуск сервера
```bash
python backend/main.py
```

### 4. Открытие сайта
- **Главная страница:** http://localhost:5000
- **Админ панель:** http://localhost:5000/admin.html

**Пароль по умолчанию:** `admin123`

1. **Для просмотра сайта** - просто откройте http://localhost:5000

- Парольная защита админ панели
- Хранение данных в localStorage (клиентская сторона)
- Отсутствие внешних зависимостей для чувствительных данных
- Локальная обработка изображений

## 📝 Разработка

### Добавление нового контента
1. Войдите в админ панель
2. Выберите раздел для редактирования
3. Внесите изменения
4. Сохраните изменения

### Расширение базы знаний
1. Добавьте новый .txt файл в папку `docs/`
2. Перезапустите сервер
3. Ассистент автоматически индексирует новый контент
